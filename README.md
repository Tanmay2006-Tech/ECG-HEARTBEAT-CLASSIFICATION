# 🫀 ECG Heartbeat Classification using Deep Learning (LSTM)

A Deep Learning-based ECG Heartbeat Classification System built using LSTM Neural Networks for detecting and classifying abnormal heartbeats from ECG signals.

---

## 🚀 Project Overview

This project uses Long Short-Term Memory (LSTM) networks to classify ECG heartbeat signals into multiple categories using publicly available ECG datasets.

The model learns temporal ECG patterns and predicts heart conditions with high accuracy.

---

## 🧠 Features

- ECG Signal Classification using LSTM
- Deep Learning Sequential Architecture
- ECG Data Preprocessing & Normalization
- Model Training & Evaluation
- Confusion Matrix Visualization
- Performance Metrics Logging
- Saved Trained Model (`.keras`)
- Training Logs & Config Files

## 📊 Dataset Used

This project uses publicly available ECG datasets:

- MIT-BIH Arrhythmia Dataset
- PTB Diagnostic ECG Dataset

These datasets are commonly used for ECG heartbeat classification and medical AI research.

## 🧪 Model Architecture

The model consists of:

- LSTM Layers
- Dense Layers
- Dropout Regularization
- Softmax Output Layer

The architecture is designed to capture sequential ECG signal patterns efficiently.
